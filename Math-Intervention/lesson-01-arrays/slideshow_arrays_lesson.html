💡 Objects in equal rows and columns = an ARRAY
Today we learn the SHORTCUT!
📖 What Is an Array?
➡️

ROW

Objects going across
left → right
"Like a racetrack!"

⬇️

COLUMN

Objects going up and down
top → bottom
"Like a skyscraper!"

🔲

ARRAY

Objects in equal rows and columns — all lined up neatly!

🏁 ROWS = racetrack (go across)  |  🏙️ COLUMNS = skyscraper (go up-down)
👀 Watch Me — I Do It First
This array → 2 rows, 4 columns
4 + 4 = 8
1

Count the ROWS ➡️

Go across — how many rows? 2 rows

2

Count ONE row ⬇️

How many in ONE row? 4 squares

3

Write the equation ✏️

Add 4 once for EACH row:
4 + 4 = 8 total

💡

That's REPEATED ADDITION!

Adding the same number again and again — one time per row.

🎵 The Array Chant!
👐
"ROWS go ACROSS!"
Slap both hands flat left → right
👆👇
"COLUMNS go UP and DOWN!"
Clap hands above head, then below waist
🔢
"Count each row — add it up!"
Point fingers counting, then pump fist on "add it up!"
⚡
"That's REPEATED ADDITION!"
Jazz hands!
🔁 Do it 3 times — get FASTER each time!
