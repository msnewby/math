Arrays & Repeated Addition

Lesson Overview

🎯 Learning Objective

Students will look at a visual array of objects arranged in equal rows and columns, identify the number of rows and columns, and write a repeated addition equation (e.g., 4 + 4 = 8) to find the total.

📌 Skill Standard

CCSS 2.OA.C.4 — Use addition to find the total number of objects arranged in rectangular arrays; write an equation to express the total as a sum of equal addends.

🔗 IEP Goal Connection

Progress monitoring shows student needs support with: identifying arrays, counting rows/columns, and representing with repeated addition — foundational for multiplication readiness.

Key Vocabulary

Array
Objects in equal rows and columns
Row
Objects going across (left → right)
Column
Objects going up and down
Equal Groups
Same number in each row
Repeated Addition
Adding the same number again and again
Total
The whole amount / answer

The 10-Minute Lesson

⏱ 10 Minutes
🔵 Hook: 1–2 min
🟠 Teach: 4–5 min
🟢 We Do: 2–3 min
1

🎣 HOOK — "Egg Carton Moment" (1–2 min)

Pull up Slide 1 on the Canva deck. Ask students to look at a picture of an egg carton (or grid of tiles, chocolate bar, etc.).

Say: "Has anyone ever seen an egg carton? How many eggs are in each row? How many rows? How do you know how many eggs total WITHOUT counting one by one? Today we're going to figure out a SHORTCUT!"

Wait for 2–3 student responses. Celebrate all answers. Connect to: "That shortcut is called repeated addition, and it uses something called an ARRAY."

2

📖 I DO — Model with an Array (2–3 min)

Use Slide 2–3. Draw or show a 2×4 array (matching the assessment problem).

SHOW THIS ARRAY → 2 rows, 4 columns
4 + 4 = 8
Say: "Watch me. I see squares. I'm going to count the ROWS first — going across — 1 row, 2 rows. So I have 2 rows. Now I'll count how many are in ONE row — 1, 2, 3, 4. So I write: 4 + 4 = 8. I added 4 TWO times — once for each row. That's REPEATED ADDITION!"
  • Point to each row as you say the number
  • Use a pointer or laser on the projected slide
  • Say "ROWS go across like a race track ➡️" and "COLUMNS go up-down like a skyscraper 🏙️"
3

🤝 WE DO — Guided Practice (2–3 min)

Show Slide 4: a new array (3×3 circles). Students write on mini whiteboards or scratch paper.

SHOW THIS → 3 rows, 3 columns (circles)
___ + ___ + ___ = ___
Say: "Your turn — work with me! How many rows? Hold up your fingers. How many in each row? Now write your equation. I'm going to walk around and peek at your boards!"

✅ Answer: 3 + 3 + 3 = 9

4

🚀 TRANSITION TO PRACTICE (30 sec)

Say: "You've got this! Now you're going to do 20 problems on your own — or with a buddy. Every time you see an array, ask yourself: HOW MANY ROWS? HOW MANY IN EACH ROW? Then add it up! Let's go!"

🎵 Optional: Start the background music playlist as students begin practice.

✏️

The 20-Minute Practice Block

⏱ 20 Minutes

📄 Practice Problem Sheet — 20 Problems

The PDF includes 20 visual array problems. Each problem has an array drawing, space to write the number of rows and columns, and an equation blank to fill in.

🖨️ Print Practice Problems (PDF) 🔑 Print Answer Key (PDF)

⏱ Timing Suggestions

  • 0–5 min: Problems 1–5 (easiest arrays)
  • 5–12 min: Problems 6–13 (medium difficulty)
  • 12–20 min: Problems 14–20 (larger arrays)

✅ Success Criteria

  • Student correctly identifies rows & columns
  • Student writes a correct addition equation
  • Mastery target: 80%+ correct (16/20)
  • Progress monitoring target: 3 sessions in a row at 80%+
🌟

Differentiation Strategies

🟠 Additional Support

  • Use physical counters to build arrays before writing
  • Pre-label rows with letters (A, B) on the worksheet
  • Provide a number line to add with
  • Allow counting one-by-one initially
  • Use graph paper to draw arrays
  • Reduce to 10 problems

🔵 On-Level (IEP Target)

  • Visual array → equation as shown
  • Identify rows and columns independently
  • Complete all 20 problems in 20 minutes
  • Self-check with answer key at end
  • Partner check after problem 10

🟣 Extension

  • Write BOTH the addition AND multiplication sentence
  • Draw their OWN array and write the equation
  • Create a word problem that matches the array
  • Try "turnaround" arrays (2×4 vs 4×2)
📊

Observation & Assessment Notes

📝 Quick Data Collection

Use tally marks during practice to track which problem types each student misses. Circle missed problems. At session end, note the total correct in your progress monitoring log.

Data to Record: Student name | Date | # Correct out of 20 | Notes on error patterns

✅ Look For (Mastery Signs)

  • Student correctly identifies rows AND columns without prompting
  • Writes the equation on the first try
  • Spontaneously uses skip-counting to add
  • Says "I have ___ groups of ___"
  • Finishes in under 20 minutes

⚠️ Watch For (Re-teach Signs)

  • Counts every single object one-by-one
  • Confuses rows and columns
  • Writes unequal addends in the equation
  • Counts columns instead of rows (wrong number of addends)
  • Cannot identify where a row starts/ends
