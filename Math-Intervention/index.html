How to Set Up Your Curriculum Folder

Keep all lesson files in one folder so every link works. The recommended structure is:
math-intervention/ → index.html (this page) + one subfolder per lesson named lesson-01-arrays/, lesson-02-next/, etc.

Put all 6 files for Lesson 1 inside lesson-01-arrays/ and all links on this card will work instantly when you open index.html in any browser.

1

Create a folder called math-intervention on your Desktop

2

Save this file as index.html inside that folder

3

Create a subfolder lesson-01-arrays and drop all 6 Lesson 1 files in it

4

Open index.html in Chrome — every link will work!

5
