🖐️ Let's Use Real Coins!
🪙

Real Coins

Use actual quarters, dimes, nickels, pennies — sort them biggest to smallest, then count up!

📏

Number Line

Start at 0. Jump +25 for quarter, +10 for dime, +5 for nickel, +1 for penny.

✏️

Write As You Go

Write each running total: 25, 35, 45, 50, 55 — don't try to add it all at once!

🔵

Counters

No real coins? Use colored counters — gold=quarter, silver=dime, brown=nickel!

🏆 Always write ¢ after your answer — 55 and 55¢ are very different things!
