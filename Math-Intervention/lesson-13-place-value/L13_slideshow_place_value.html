📖 Words We Need to Know
Hundreds Place
The LEFT digit of a 3-digit number
794 → 7 hundreds
Tens Place
The MIDDLE digit of a 3-digit number
794 → 9 tens
Ones Place
The RIGHT digit of a 3-digit number
794 → 4 ones
Digit
A single number symbol: 0, 1, 2, 3, 4, 5, 6, 7, 8, or 9
Place Value
A digit's VALUE depends on WHERE it sits in the number
